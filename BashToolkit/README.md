# System Administration Bash Toolkit

**Author:** Eyasu Solomon  
**Language:** Bash  
**Platform:** Linux/Unix/Git Bash

## Overview

A comprehensive Bash scripting toolkit for system administration tasks. Demonstrates proficiency in shell scripting, automation, and IT operations.

## Features

- **System Information** - OS, hardware, and user details
- **Disk Usage Analysis** - Visual disk usage with color-coded warnings
- **Network Diagnostics** - Interface info, DNS, connectivity tests
- **Process Manager** - Top CPU/memory consumers
- **Service Manager** - View running/failed services
- **Log Analyzer** - Parse logs for errors and warnings
- **Backup Utility** - Create timestamped tar.gz backups
- **Security Audit** - User accounts, SSH config, open ports
- **System Cleanup** - Clean temporary files

## Usage

### Interactive Mode
```bash
chmod +x sysadmin_toolkit.sh
./sysadmin_toolkit.sh
```

### Command Line Mode
```bash
./sysadmin_toolkit.sh sysinfo     # System information
./sysadmin_toolkit.sh disk        # Disk usage
./sysadmin_toolkit.sh network     # Network diagnostics
./sysadmin_toolkit.sh process     # Process manager
./sysadmin_toolkit.sh service     # Service manager
./sysadmin_toolkit.sh log         # Log analyzer
./sysadmin_toolkit.sh backup      # Backup utility
./sysadmin_toolkit.sh security    # Security audit
./sysadmin_toolkit.sh cleanup     # System cleanup
```
